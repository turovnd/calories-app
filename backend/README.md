# Backend for calories app

## `.env` config
```
SERVER_PORT=3100
MONGO_URL=
MONGO_USER=admin
MONGO_PASSWORD=admin
MONGO_HOST=localhost:27017
MONGO_DATABASE=calories?authSource=admin
TOKEN_HASH=random-1
PASSWORD_HASH=random-2
```

## How to

### Start locally
```
npm install
npm run dev
```

### Build
```
npm install
npm run build
```
